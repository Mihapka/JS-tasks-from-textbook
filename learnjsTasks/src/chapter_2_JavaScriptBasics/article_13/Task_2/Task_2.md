# Замените for на while                                                     <br/>

Перепишите код, заменив цикл for на while, без изменения поведения цикла.   <br/>                                                                   
for (let i = 0; i < 3; i++) {                                               <br/>
alert( `number ${i}!` );                                                    <br/>
}                                                                           <br/>
