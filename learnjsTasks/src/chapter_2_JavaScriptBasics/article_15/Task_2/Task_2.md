# Функция min(a, b)                                                     <br/>

Напишите функцию min(a,b), которая возвращает меньшее из чисел a и b.   <br/>

Пример вызовов:                                                         <br/>

min(2, 5) == 2                                                          <br/>
min(3, -1) == -1                                                        <br/>
min(1, 1) == 1                                                          <br/>
